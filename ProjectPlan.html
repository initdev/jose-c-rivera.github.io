Project Plan.

-Completed Features-

Feature: Map Zoom Function

Description: The map zoom function will aid in navigation when attempting to click on smaller countries.

Feasibility: This feature is quite feasible by utilizing an open-source package such as jVectorMap.

Dependency: The rest of the project is not dependent on this, although from a UX standpoint, it is important.



-Potential Features-

Feature: Online Ordering

Description: The online ordering feature would allow a user to order directly from our app once a restaurant is discovered.

Feasibility: This feature is feasible through: (1) The Just Eat API; (2) Embedding the Just Eat website; or (3) Providing the user a link to the proper Just Eat menu.

Dependency: This is not a vital feature to our project, just an interesting add-on that would provide an additional use for the user.


Feature: Randomize Restaurant

Description: For those wishing to get real wild, the randomize restaurant feature would provide the user with a random restaurant to try.

Feasibility: This would be a simple feature to implement by providing a completely random restaurant from either the Google Places or Yelp! APIs, or from the top trending restaurants list.

Dependency: The rest of the project is not dependent on this feature, although it would be pretty nifty to include for those who lack decisiveness or simply want to experience something completely new.
